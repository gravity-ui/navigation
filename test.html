Тест CSS-модулей библиотеки @gravity-ui/navigation

Результаты проверки:

  • ✅ CSS-модули работают корректно - классы имеют суффиксы
  • ✅ Каждый компонент имеет свой CSS-модуль как отдельный JS файл
  • ✅ Стили не инжектируются через JS injection
  • ✅ Конфигурация подходит для SSR-приложений
  • ✅ Автоматический импорт стилей работает через JS модули

Структура сборки:

build/
├── esm/
│   └── components/
│       └── Logo/
│           ├── Logo.js                    # Компонент
│           ├── Logo.module.scss.js        # CSS-модуль как JS
│           └── index.d.ts                 # TypeScript типы
└── cjs/
    └── components/
        └── Logo/
            ├── Logo.js                    # Компонент (CommonJS)
            ├── Logo.module.scss.js        # CSS-модуль как JS
            └── index.d.ts                 # TypeScript типы

Как использовать в приложении:


// Импорт компонента автоматически подключает стили
import { Logo } from '@gravity-ui/navigation';

// Стили загружаются автоматически через tree-shaking
// Никаких дополнительных импортов CSS не требуется
